HOMEWORK 6 - GRADING KEY

1. (17 points) Documentation and Coding Style

  • Name, ID number and recitation at the top of ALL java files [2 points]
  • Precise and correct specifications. [5 points]
  • Indentation. [3 points]
  • Variable Names. [2 points]
  • Javadoc Style for Documentation. [5 points]

Note: You only need to submit your java files. We will use the javadoc *.java command to create the javadoc files.

Note 2: DO NOT SUBMIT bigdata.jar along with your Java source files, otherwise 10 points will be deducted.

2. (13 Points) Source Code

  • Implementation done as specified (additional methods are OK) [8 points]
  • Throw an exception back to the calling method(s) if a precondition is violated. [5 points]

3. (70 Points) Program Correctness

  • Program compiles without any errors. [12 points]
  • The following menu options are implemented correctly.:
    • (D) - Import Data from URL [10 points]
    • (A) - Create a New Auction [5 points]
    • (B) - Bid on an Item [4 points]
    • (I) - Get Info on Auction [4 points]
    • (P) - Print All Auctions [8 points]
    • (R) - Remove Expired Auctions [8 points]
    • (T) - Let Time Pass [3 points]
    • (Q) - Quit [1 points]
  • AuctionTable class can be saved to/loaded from a file [10 points]
  • Erroneous input handled "gracefully." (Appropriate Error messages displayed to the user.) [5 points]

Note: DO NOT SUBMIT bigdata.jar along with your Java source files, otherwise 10 points will be deducted.
